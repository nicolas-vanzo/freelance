💻 Sobre o projeto
A ideia central é um marketplace de serviços especializados (como IA, Cibersegurança e DevOps). O usuário pode navegar pelas categorias, visualizar os valores por hora e gerenciar um carrinho de pedidos que calcula o total em tempo real.

O projeto foca na clareza visual e na execução direta de scripts para manipulação do DOM (Document Object Model).

🛠️ O que foi utilizado
Para manter a leveza e a performance, o projeto foi construído puramente com as tecnologias fundamentais da web:

HTML5: Estruturação semântica de seções, cabeçalho e rodapé.

CSS3: Estilização baseada em Flexbox para garantir que os cards de serviços fiquem organizados.

JavaScript (Vanilla): Lógica de programação para captura de cliques, cálculos matemáticos de preços e atualização dinâmica da lista de pedidos sem necessidade de recarregar a página.

📂 Estrutura de Arquivos
index.html: Contém a estrutura da página, os textos dos serviços e os botões de ação.

style.css: Define a identidade visual, as cores (tons de cinza e branco), espaçamentos e os efeitos de hover nos botões.

script.js: O "cérebro" da aplicação. Ele monitora qual botão foi clicado, extrai o nome e o valor do serviço e atualiza o total acumulado.
